# AI Agent Chat Application

A full-stack chat application for tuition queries and payments, using React (frontend), Node.js API Gateway (backend), and Firestore for real-time messaging.

## Folders
- `frontend/` — React chat UI
- `backend/gateway/` — Node.js API Gateway
- `backend/firebase/functions/` — Cloud Functions for LLM intent parsing

## How to Run
1. Install dependencies in both `frontend` and `backend/gateway`.
2. Start the backend gateway: `npm start` in `backend/gateway`.
3. Start the frontend: `npm start` in `frontend`.
4. (Optional) Set up Firestore and Cloud Functions for real-time messaging and LLM integration.

## Features
- Chat UI for tuition queries and payments
- API Gateway routing all API calls
- Firestore for real-time chat (to be integrated)
- LLM intent parsing (to be integrated)
